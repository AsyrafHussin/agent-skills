# Rule Sections

## Priority Levels

| Level | Description | When to Apply |
|-------|-------------|---------------|
| CRITICAL | Essential for any TS+React project | Always |
| HIGH | Common patterns used frequently | Most components |
| MEDIUM | Advanced patterns for specific needs | When building reusable components |
| LOW | Specialized utilities | Edge cases |

## Section Overview

### Component Typing (CRITICAL)
Fundamental patterns for typing React component props. Every component needs these.

### Hook Typing (CRITICAL)
Patterns for typing React hooks. Essential for useState, useRef, and custom hooks.

### Event Handling (HIGH)
Correct event types for form inputs, buttons, and other interactive elements.

### Ref Typing (HIGH)
Patterns for useRef with DOM elements and mutable values.

### Generic Components (MEDIUM)
Building type-safe, reusable components with TypeScript generics.

### Context & State (MEDIUM)
Creating and consuming typed React context.

### Utility Types (LOW)
React and TypeScript utility types for advanced patterns.
