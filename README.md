# Agent Skills

A collection of skills for AI coding agents, designed for Claude Code.

## Available Skills

| Skill | Description |
|-------|-------------|
| react-vite-best-practices | React + Vite performance optimization |
| typescript-react-patterns | TypeScript best practices for React |
| laravel-best-practices | Laravel 12 conventions |
| laravel-inertia-react | Laravel + Inertia.js + React patterns |
| tailwind-best-practices | Tailwind CSS patterns |
| state-management | React Query + Zustand patterns |
| web-design-guidelines | UI/UX and accessibility |
| php-best-practices | PHP 8.x, PSR, SOLID |
| clean-code-principles | SOLID, DRY, design patterns |
| api-design-patterns | RESTful API design |
| git-workflow | Commits, branching, PRs |
| testing-best-practices | Unit testing, TDD |

## Installation

```bash
git clone https://github.com/AsyrafHussin/agent-skills.git
cp -r agent-skills/skills/* ~/.claude/skills/
```
